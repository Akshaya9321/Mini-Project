# Mini-Project
A smart attendance tracking system to replace error-prone manual methods with automated, transparent, and proxy-proof records.
Smart Attendance System

Problem

Manual attendance is time-consuming and can lead to errors and proxy attendance.

Solution

The Smart Attendance System automates attendance tracking and provides accurate records for students and teachers.

Features

- Digital attendance marking
- Real-time attendance tracking
- Automated attendance reports
- Reduced manual work
- Improved accuracy

Technologies Used

- Python
- HTML
- CSS
- JavaScript
- MySQL

Future Enhancements

- QR Code Attendance
- Face Recognition
- Mobile Application Support

Conclusion

This project helps educational institutions manage attendance efficiently and accurately.
